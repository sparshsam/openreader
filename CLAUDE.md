# PDF Reader by Sparsh — Claude Code Instructions

## Overview

Private, local-first desktop PDF tool for reading, annotation, search, diff, and workspace restore.

## Commands

Refer to project Makefile/build scripts for build instructions.

## Architecture Constraints

1. Local-first. No cloud dependency.
2. Privacy by design.
3. Cross-platform (Windows primary, macOS secondary).

## Workflow

1. Branch from \`main\`.
2. PR for every merge. No direct pushes to \`main\`.
