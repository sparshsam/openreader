# v0.3.2 to v0.3.3 GUI Updater Validation Test

## Prerequisites

- You must have v0.3.2 installed on your Windows machine.
- You must have a working internet connection (GitHub API access).
- The v0.3.3 release must be published.

## Test Steps

1. Locate your v0.3.2 installation
2. Launch v0.3.2
3. Verify current version shows 0.3.2
4. Click Check for Updates - should detect v0.3.3
5. Observe download of PDFReader-by-Sparsh-Windows.zip
6. Verify NO update_None error appears
7. Let the update apply (auto-close, extract, replace, relaunch)
8. Verify post-update version shows v0.3.3

## If the update fails

Provide Sparsh with:
1. Screenshot of the error/message
2. updater-debug.log from %TEMP%\PDFReader-Updates\
3. The launch directory

## Updater Log Location

%TEMP%\PDFReader-Updates\updater-debug.log

Opens via:
notepad "%TEMP%\PDFReader-Updates\updater-debug.log"
