# Privacy Policy

**Application:** OpenReader — a local-first desktop PDF utility for Windows.

**Effective date:** June 18, 2026

See the rendered policy at:
**[https://sparshsam.github.io/openreader/privacy/](https://sparshsam.github.io/openreader/privacy/)**

---

## Data Collection

**OpenReader does not collect, transmit, or store any personal data.**

The application is entirely local-first. All PDF processing — reading, searching, copying, annotating, merging, splitting, extracting, and compressing — occurs on your local machine. No document content is ever uploaded, transmitted, or shared.

## Network Access

OpenReader makes one optional network request:

- **Update check (opt-in):** On launch (if enabled), the app sends a GET request to the GitHub Releases API to compare versions. This includes no identifiers, document data, or telemetry.

No other network requests are made. No analytics, crash reporting, or usage tracking is included.

## Local Storage

OpenReader stores preferences, session state, and optional search indexes locally in your user profile. This data never leaves your device.

## Third-Party Services

No third-party analytics or tracking SDKs are embedded. The only external service is GitHub (update check) and the Microsoft Store (distribution).
